# Finance Track

Controle financeiro pessoal com dashboard, autenticação e relatórios.

## Tecnologias

- **Frontend:** Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** NestJS, PostgreSQL, Prisma, JWT
- **Infra:** Docker, Railway, Vercel

## Como rodar

```bash
# Frontend
cd web
npm install
npm run dev

# Backend
cd api
npm install
npm run start:dev
```

## Funcionalidades

- [ ] Autenticação (login e cadastro)
- [ ] Dashboard com resumo do mês
- [ ] Cadastro de transações (receitas e despesas)
- [ ] Categorias personalizadas
- [ ] Gráficos de gastos
